# AI Agent Suite

This project is a combination of several AI-powered agents designed to interact with documents and assist with tasks such as quiz generation and job searching.

## Requirements

Install LLaMA and select the Ollama 3.2 model.

Install the required Python packages:

pip install -r requirements.txt

## quiz.py

This module is designed to interact with PDF documents. It enables features such as generating quizzes and answering questions based on the content of a given PDF.

### Usage

Run the following command:

python quiz.py

## job.py

This module helps in finding jobs online based on your portfolio and PDF. It analyzes your documents and searches for relevant job opportunities based on your data.

### Usage

Run the following command:

python job.py
